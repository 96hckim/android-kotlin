# Function
 * Retrofit 을 이용하여 재생 목록을 받아와 구성
 * 재생 목록을 클릭하여 ExoPlayer 통해 음악 재생
 * 이전, 다음 트랙 버튼을 눌러서 이전, 다음 음악을 재생하고 UI 업데이트
 * PlayList 화면과 Player 화면 간의 전환할 수 있다
 * SeekBar 를 Custom 하여 원하는 UI 로 표시할 수 있다
